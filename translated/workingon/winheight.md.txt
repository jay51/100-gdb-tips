#Window size

## example

``` 
#include <stdio.h>

void fun1(void) {
int i u003d 0;

i++;
i u003d i * 2;
printf("%d\n", i);
}

void fun2(void) {
int j u003d 0;

fun1();
j++;
j u003d j * 2;
printf("%d\n", j);
}

int main(void) {
fun2();
return 0;
}
```


## Tips

When using gdb graphical debugging interface, you can use the "`winheight <win_name> [+ | -]count`" command to adjust the window size (`winheight` is abbreviated as `win`. `win_name` can be `src`, `cmd` , `asm` and `regs`). Take the debug above as an example. This is the original `src` window size:
